A design system is a collection of reusable components and guidelines that ensures consistency across all digital products.

Design systems contain colors, fonts, buttons, icons, spacing rules, and other visual elements that teams use to build websites and apps. Thy ensure that every page, screen, and feature looks and works the same way, creating a cohesive user experience. On this website, the DUX team has made sure everything is documented with clear examples of how and when to use each piece.
